Hướng dẫn chạy project:
+ Sử dụng XAMPP mở phpMyAdmin và tạo database tên sql_java
+ Cài đặt IntelliJ, vào file application.properties sửa lại username và password (mặc định là username="root", password ="")
+ Mở file FinaltermApplication.java, tiến hành chạy project bằng IntelliJ
+ Lưu ý: Khi chạy lần đầu sẽ sinh ra nhiều Warning do chưa có tables trong database
+ Chạy project lần 1 sẽ sinh ra các tables trong database
+ Chạy project lần 2 để hoàn tất việc chạy server
+ Mở VSCode để chạy FE bằng file dangnhap.html và đăng nhập bằng tài khoản được cung cấp
